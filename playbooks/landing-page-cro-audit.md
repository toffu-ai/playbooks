1. Scrape the target landing page content and structure
2. Evaluate against CRO criteria: headline clarity, value prop, social proof, CTA placement, form friction, page speed signals
3. Pull GA4 data for that page: bounce rate, time on page, conversion rate
4. Score each criterion and rank issues by estimated conversion impact
5. Output a prioritized fix list with specific recommendations for each issue
