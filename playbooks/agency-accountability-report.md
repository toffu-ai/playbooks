Track exactly what my agency changed in my Google Ads account, cross-reference with performance, and hold them accountable.

1. Pull all changes from my Google Ads account from the past 7 days, grouped by who made each change
2. Categorize changes by type: bid adjustments, budget changes, keyword additions/removals, ad copy edits, audience modifications, campaign structure changes
3. For each major change, compare the old and new values
4. Pull performance metrics from Google Ads and Google Analytics for the same period to correlate changes with results:
   - Did the bid change improve or hurt CPA?
   - Did the new keywords drive conversions or just spend?
   - Did the audience changes affect conversion rate?
5. Build a Google Sheet report with:
   - Change log by person (who did what, when)
   - Change impact analysis (performance before vs after)
   - Flagged changes that correlated with performance drops
   - How each change was made (manually in the UI, via API, via scripts, or auto-applied by Google)
6. Email me the weekly agency accountability report
7. Schedule weekly execution
