1. List the conversion events that matter: signup, purchase, demo request, free trial start
2. Check which events are already tracked in GTM vs. GA4
3. Create GTM tags and triggers for any missing conversion events
4. Link each tag to the corresponding GA4 key event
5. Publish the GTM container and verify events fire correctly in GA4 DebugView
