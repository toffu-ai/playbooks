1. Fetch the blog post content from a URL or CMS
2. Extract the core argument, 3 key points, and the actionable takeaway
3. Write a 60-90 second video script: strong hook, 3 punchy sections, clear CTA
4. Add scene notes and on-screen text suggestions for each section
5. Save the script to Google Docs, ready to send to the video team
